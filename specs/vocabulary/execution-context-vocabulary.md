# Execution Context Ontology Vocabulary

**Namespace:** `https://ns.slashlife.ai/agent/execution-context`

Defines the contextual frame for AI Agent actions, including environment, security, and resource constraints.

## Classes

### `Budget`

**IRI:** `https://ns.slashlife.ai/agent/execution-context#Budget`

A constraint on the financial expenditure for a task.

### `ExecutionEnvironment`

**IRI:** `https://ns.slashlife.ai/agent/execution-context#ExecutionEnvironment`

Describes the host environment where an action is executed.

### `InputState`

**IRI:** `https://ns.slashlife.ai/agent/execution-context#InputState`

Represents the inputs provided to an action.

### `OutputState`

**IRI:** `https://ns.slashlife.ai/agent/execution-context#OutputState`

Represents the outputs returned by an action.

### `RateLimit`

**IRI:** `https://ns.slashlife.ai/agent/execution-context#RateLimit`

A constraint on the frequency of an action or request.

### `ResourceConstraint`

**IRI:** `https://ns.slashlife.ai/agent/execution-context#ResourceConstraint`

A constraint on a computational resource (e.g., memory, CPU, storage).

### `SecurityContext`

**IRI:** `https://ns.slashlife.ai/agent/execution-context#SecurityContext`

Security-related constraints and guarantees active during execution.
