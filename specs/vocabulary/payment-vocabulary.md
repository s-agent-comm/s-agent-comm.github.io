# Payment Ontology Vocabulary

**Namespace:** `https://ns.slashlife.ai/agent/payment`

Defines concepts for expressing payment intents and settlement interfaces between AI Agents.

## Classes

### `Payment Intent`

**IRI:** `https://ns.slashlife.ai/agent/payment#PaymentIntent`

A semantic expression of an intent to transfer value from one agent to another.

### `Settlement Interface`

**IRI:** `https://ns.slashlife.ai/agent/payment#SettlementInterface`

Describes the technical and semantic requirements for settling a payment.
